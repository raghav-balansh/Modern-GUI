## Installation
To run this project, you need to install the required modules. You can install all dependencies at once using the `requirements.txt` file generated for this project.

```bash
pip install -r requirements.txt
```

### Manual Installation
If you prefer identifying specific modules, here is the list:
```bash
pip install psutil comtypes pycaw screen-brightness-control geopy timezonefinder==6.0.1 requests pytz tkcalendar speedtest-cli Pillow PyAutoGUI numpy==1.26.4
```
**Note:** `speedtest` module is provided by `speedtest-cli`. `numpy` must be compatible with `timezonefinder` (version < 2.0).

If it not work go to the web browser and searches new module and run on that kind of modules.
